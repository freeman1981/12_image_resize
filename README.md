# Image Resizer
## Installation
You should install Pillow.
```python3 -m pip install -r requirements.txt```

It is better use virtual enviroments
```python3 -m venv venv```
## How to use
```python3 image_resize.py --help```
will show options.
You should use --scale or --width and --height
